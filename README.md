# EUR/USD Elliott RSI

A Python research scaffold for EUR/USD strategies that combine Elliott-wave-inspired swing features, RSI indicators, supervised/unsupervised models, and a lightweight vectorized backtesting engine.

## Layout

- `src/eurusd_elliott_rsi/data/` - market data adapters.
- `src/eurusd_elliott_rsi/features/` - technical feature generation.
- `src/eurusd_elliott_rsi/strategies/` - signal generation.
- `src/eurusd_elliott_rsi/models/` - supervised and unsupervised ML helpers.
- `src/eurusd_elliott_rsi/backtest/` - custom vectorized backtesting engine.
- `configs/` - experiment configuration examples.
- `tests/` - smoke tests for the scaffold.
